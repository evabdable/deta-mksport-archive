# deta-mksport-archive

## 简介

`deta-mksport-archive` 是一个用于归档和发布多个独立 HTML 页面的仓库。  
本仓库不针对任何单个域名或具体网站，仅作为 HTML 文件的集中存储与展示用途。

所有页面均为静态内容，可直接通过 GitHub Pages 或其他静态托管服务访问。

## 目录结构

```
deta-mksport-archive/
├── index.html          # 仓库主页 / 导航页
├── pages/              # 存放独立的 HTML 页面
│   ├── page1.html
│   ├── page2.html
│   └── ...
├── assets/             # 静态资源（图片、CSS、JS 等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md
```

- `index.html`：仓库的入口页面，可提供页面列表或简介。
- `pages/`：归档的各个独立 HTML 页面，按需组织存放。
- `assets/`：页面所需的共用资源文件。

## 页面归档说明

- 每个 HTML 页面均为独立文件，可直接在浏览器中打开运行。
- 页面之间可能共享部分资源（如样式、脚本），统一放置在 `assets/` 目录下。
- 归档内容以原始状态保存，不保证长期兼容所有浏览器版本。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接在浏览器中打开 `index.html` 或 `pages/` 下的任意 HTML 文件。
3. 若需部署，可将整个仓库上传至静态托管服务（如 GitHub Pages、Netlify 等）。

- 不保证所有页面长期可用或内容完整。
- 如发现页面异常或资源缺失，可提交 Issue 反馈。
